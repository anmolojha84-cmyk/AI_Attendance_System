# 🎓 AI Smart Attendance Monitoring System

An AI-powered Smart Attendance Monitoring System developed using **Python**, **OpenCV**, and **Face Recognition** technology. The system automatically detects registered students, marks attendance in real time, provides voice confirmation, generates attendance reports, and stores unknown faces separately.

---

## 📌 Features

✅ Face Recognition Based Attendance

✅ Real-Time Student Detection

✅ Automatic Attendance Marking

✅ Voice Confirmation

✅ Unknown Face Detection

✅ Attendance Report (CSV/PDF)

✅ AI Parent Email Generator

✅ Face Encoding Generation

✅ Modern GUI Dashboard

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/AI_Attendance_System.git
```

Move into the project

```bash
cd AI_Attendance_System
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python gui.py
```

## 🚀 Workflow

1. Register Student
2. Capture Face Image
3. Generate Face Encodings
4. Start Attendance
5. Face Recognition
6. Attendance Marked Automatically
7. Voice Confirmation
8. CSV/PDF Report Generated
